# Special Cases & Edge Cases

## 1. Idioms, Deyimler, Expresiones, Ausdrücke

### Strategy: Find Equivalent, Never Literal

**Golden Rule:** Idioms don't translate word-for-word. Find the meaning equivalent in target language.

## 2. Humor, Irony & Sarcasm

### Strategy: Preserve Effect, Not Words

Humor is the **hardest** to translate because it relies on:
- Cultural context
- Sound/rhythm
- Unexpected juxtaposition
- Language-specific wordplay

### Rules for Humor Translation

**Do:**
- ✅ Preserve the **effect** (should still be funny/ironic)
- ✅ Use target language's humor tools
- ✅ Consider cultural comedy styles
- ✅ Note if humor doesn't translate
- ✅ Offer alternative that works in target language

**Don't:**
- ❌ Translate jokes literally (they'll be unfunny)
- ❌ Add explanation (ruins humor)
- ❌ Change the point to make it "funnier"
- ❌ Assume humor translates directly

### Turkish Humor Tools

- "Tabii ki..." (of course...) — sarcastic tone
- "Sanki..." (as if...) — ironic framing
- "Tam da ihtiyacım olan şey" (exactly what I needed) — sarcastic
- "Ne güzel işte" (how wonderful) — sarcastic
- Üç nokta (...) — implies meaning unsaid
- Exaggeration for effect

**Example:**
```
English (humorous): "Oh great, another meeting that could have been an email."
❌ Literal Turkish: "Oh harika, bir email olabilecek başka bir toplantı daha."
✅ Turkish with Humor: "Harika, bir e-posta ile halledilebilecek bir toplantı daha. Tam da ihtiyacım olan şey."
```

### Spanish Humor Tools

- Exaggeration (¡qué drama!)
- Diminutives/augmentatives for tone
- Ironic expressions ("¡Qué maravilla!")
- Regional humor styles vary

### French Humor Tools

- Subtle wit and irony
- Understatement (litote)
- Wordplay on homophones
- Intellectual humor tradition

### German Humor Tools

- Directness & bluntness
- Compound word wordplay
- Exaggeration
- Sarcastic emphasis through intonation (reflected in punctuation)

## 3. Cultural References

### Decision Tree for References

| Situation | Action | Example |
|-----------|--------|---------|
| **Universal pop culture** | Keep reference | "Star Wars" → "Star Wars" |
| **Western-specific, no Turkish equivalent** | Explain or adapt | "Super Bowl" → "Super Bowl (ABD futbolunda final)" |
| **Western-specific, Turkish equivalent exists** | Adapt | "Like finding needle in haystack" → "Samanlıkta iğne aramak" |
| **Culturally meaningless in target** | Localize | "$5 coffee" → "50 liralık kahve" (context-dependent) |
| **Famous person/author** | Keep name, provide context if needed | "Shakespeare said..." → "Shakespeare şöyle dedi..." |
| **Holiday/celebration** | Adapt to local equivalent or explain | "Thanksgiving" → "Thanksgiving (ABD'de Şükran Günü)" |

### Turkish Cultural Adaptations

| Reference | Keep As-Is | Explain | Localize |
|-----------|-----------|---------|----------|
| "American as apple pie" | No | No | "Türk kahvesi kadar Türk" |
| "Merry Christmas" | Depends | Maybe | "Mutlu Noeller" (Christmas) or keep |
| "Thanksgiving" | No | Yes - explain holiday | Localize to Turkish context |
| "Black Friday" | Possibly | Yes | "Kara Cuma indirimleri" |
| "Nobel Prize" | Yes | Maybe | Keep as-is (universal) |

### Spanish Cultural Adaptations

| Reference | Approach |
|-----------|----------|
| "Thanksgiving" | Explain or omit (not celebrated in Spanish-speaking countries) |
| "Super Bowl" | Explain or adapt (not primary sport interest) |
| "Halloween" | Less cultural weight in Spain/LATAM - explain if important |
| "Soccer/Football" | Universal - keep, or adapt terminology |

### French Cultural Adaptations

| Reference | Approach |
|-----------|----------|
| "American Thanksgiving" | Not celebrated in France - explain |
| "French fries" | French prefer "frites" or just "frites" (ironic) |
| Famous French figures | Localize knowledge naturally |

### German Cultural Adaptations

| Reference | Approach |
|-----------|----------|
| "Oktoberfest" | Universal reference - keep |
| "German beer" | Universal stereotypes - can use |
| "Wurst" | Use German term (commonly known) |

## 4. Numbers, Measurements & Money

### Money Conversion Strategy

**Rule:** Adapt to local context. Don't leave foreign currency if irrelevant.

| Original | Context | Turkish | Spanish | French | German |
|----------|---------|---------|---------|--------|--------|
| "$100" | General | "100 dolar" | "100 dólares" | "100 dollars" | "100 Dollar" |
| "$5 coffee" | Relatable | "50 liralık kahve" | "un café de $5" | "un café de 5 dollars" | "einen Kaffee für 5 Dollar" |
| "£20" | Specific | "20 sterlin" | "20 libras" | "20 livres" | "20 Pfund" |

### Measurement Conversion

**Rule:** Use metric system in target language countries (most of world). Keep original if technical.

| Original | Turkish | Spanish | French | German |
|----------|---------|---------|--------|--------|
| "5 miles" | "8 kilometre" | "8 kilómetros" | "8 kilomètres" | "8 Kilometer" |
| "6 feet tall" | "1,80 metre boyunda" | "1,80 metros de altura" | "1,80 mètres" | "1,80 Meter groß" |
| "32°F" | "0°C" | "0°C" | "0°C" | "0°C" |

**Exception:** If text is technical specification, keep original units.

### Date & Time Formats

## 5. Addressing & Pronouns (sen/siz, tú/usted, tu/vous, du/Sie)

### Decision Tree for Formality Level

```
Is the text addressing the reader directly?

YES → Continue below
NO → Skip, use passive or general statements

Is it formal/professional?
├── YES → Use formal you: siz/usted/vous/Sie
├── NO → Continue

Is it casual/friend?
├── YES → Use informal you: sen/tú/tu/du
└── NO → Ambiguous (default to formal)
```

### Consistency Rule (CRITICAL)
**Pick one. Stick with it throughout the entire document.**

Mixing "sen" and "siz" in same document = **jarring for reader**.

### Turkish Addressing

| Context | Use | Example |
|---------|-----|---------|
| Technical documentation | "siz" (formal) | "Başlık üzerine tıklayınız" |
| Blog post / casual | "sen" (informal) | "Başlığa tıkla" |
| User agreement / legal | "siz" (formal) | "Şartları kabul etmekte misiniz?" |
| Friend/personal letter | "sen" (informal) | "Seni özledim" |
| General statement | Avoid pronoun | "Bu adım önemli" (instead of "Siz/sen bu adımı yapmalısınız") |

### Spanish Addressing

| Context | Use | Example |
|---------|-----|---------|
| Professional | "usted" (formal) | "Haga clic en el título" |
| Casual | "tú" (informal) | "Haz clic en el título" |
| Spain (friends) | "vosotros" (group informal) | "Haced clic en el título" |
| Legal/formal | "usted" | "¿Acepta los términos?" |

### French Addressing

| Context | Use | Example |
|---------|-----|---------|
| Professional | "vous" (formal) | "Cliquez sur le titre" |
| Formal written | "vous" | "Vous devez cliquer..." |
| Casual/friend | "tu" (informal) | "Clique sur le titre" |
| Group formal | "vous" | "Vous êtes invités..." |

### German Addressing

| Context | Use | Example |
|---------|-----|---------|
| Professional | "Sie" (capitalized, formal) | "Klicken Sie auf den Titel" |
| Business | "Sie" | "Sie müssen bestätigen..." |
| Casual/friend | "du" | "Klick auf den Titel" |
| Group formal | "Sie" | "Sie sind eingeladen..." |

## 6. What NOT to Translate (Universal)

**Never translate:**
- Code blocks (code stays in original language)
- URL/web addresses
- Email addresses
- Brand/product names (usually)
- Acronyms (HTML, API, CEO, etc.) — unless established translation exists
- Proper names (personal names, place names)
- Formulas and equations
- File names and paths

**Example:**
```
❌ "URL'yi git https://example.com/dokumanlar/giriş adresine"
✅ "URL'yi git https://example.com/documents/intro adresine"

❌ Code: "const myFunction = () => { console.log('Merhaba'); }"
✅ Code: "const myFunction = () => { console.log('Hello'); }" (don't translate string literals)
```

## 7. Ambiguity Resolution

When original text is ambiguous or unclear:

**Do:**
- ✅ Make reasonable choice based on context
- ✅ Translate the most likely interpretation
- ✅ Add note: "[Translator note: Original was ambiguous...]"
- ✅ Flag if meaning could change understanding

**Don't:**
- ❌ Translate both interpretations (confusing)
- ❌ Add explanation without being asked
- ❌ Ignore the ambiguity and guess wrong

**Example:**
```
Ambiguous original: "The bank is near the post office."
(Could mean: financial bank OR river bank)

Turkish translation: "Banka posta dairesi yakınında." (assumed financial bank)
Note: [Original metinde "bank" sözcüğü öncüsüz; finansal banka olarak tercih edildi.]
```
